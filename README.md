Market Intelligence Pipeline

This project is a full-stack web application that I built to automatically analyze the job market. It uses an undetected browser to scrape live job listings and sends this data to the Gemini 2.5 Flash API to extract the most in-demand technical skills.

I developed the frontend using Next.js and Tailwind CSS for a modern interface, while the backend is powered by Python and FastAPI. To run the project, you just need to install the requirements, start the FastAPI backend using Uvicorn, and run the Next.js frontend with the npm run dev command.

Pazar Analiz Aracı

Bu proje, iş ilanlarını otomatik olarak analiz etmek için geliştirdiğim full-stack bir web uygulamasıdır. Arka planda çalışan hayalet bir tarayıcı ile canlı iş ilanlarını toplar ve piyasada en çok aranan teknik yetenekleri bulmak için bu verileri Gemini 2.5 Flash API'ye gönderir.

Kullanıcı arayüzünü Next.js ve Tailwind CSS ile hazırladım, arka uç kısmını ise Python ve FastAPI kullanarak yazdım. Projeyi çalıştırmak için gerekli kütüphaneleri kurduktan sonra, backend tarafını Uvicorn ile, frontend tarafını ise npm run dev komutuyla başlatmak yeterlidir.
